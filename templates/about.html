Acerca del Sistema

Este sistema de ventas fue desarrollado para simplificar la gestión de inventario y ventas en supermercados. Permite agregar, actualizar y consultar productos de manera rápida y segura.

Utiliza tecnologías modernas como Flask, Bootstrap 5 y JavaScript para ofrecer una experiencia interactiva y profesional al usuario. Ideal para pequeños y medianos negocios.
